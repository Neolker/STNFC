Release Notes for X-CUBE-NFC6 Projects

Copyright © 2022 STMicroelectronics

Purpose

The X-CUBE-NFC6 software package is an expansion for STM32Cube. This software provides drivers running on STM32 for ST25R3916/ST25R3916B device.
